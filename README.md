# Tic-Tac-Toe React 🕹️

**Tic-Tac-Toe React**, popüler "X ve O" oyununun modern web teknolojileriyle geliştirilmiş, interaktif bir versiyonudur. Bu proje, React'in state yönetimi, component yapısı ve oyun mantığı algoritmarını sergilemek amacıyla geliştirilmiştir.

---

## ✨ Özellikler

* **Dinamik Oyun Akışı:** Oyuncular (X ve O) arasında otomatik geçiş.
* **Akıllı Kazanma Algoritması:** Her hamlede satır, sütun ve çapraz kontrolü yaparak kazananı anında belirler.
* **Beraberlik Kontrolü:** Tüm kareler dolduğunda ve kazanan olmadığında oyunu bitirir.
* **Modern UI:** CSS ile güçlendirilmiş, sade ve şık kullanıcı arayüzü.
* **Performanslı Yapı:** Vite ile hızlı geliştirme ve derleme süreci.

---

## 🛠️ Kullanılan Teknolojiler

* **React:** UI bileşenleri ve state yönetimi için.
* **Vite:** Hızlı ve modern bir frontend build aracı.
* **JavaScript (ES6+):** Temel oyun mantığı ve algoritmalar.
* **CSS3:** Responsive ve temiz bir tasarım.
